Funciones principales

Esta página sirve como guía para las funciones principales de la aplicación

Esta aplicación está destinada al manejo de los clientes de un sistema

Sus funciones principales son:

  1. Añadir cliente
  2. Modificar cliente
  3. Borrar cliente

También se está trabajando en algunas funciones experimentales como:

  1. Verificar cliente
  2. Suspender cliente temporalmente
  3. Mejorar cliente
