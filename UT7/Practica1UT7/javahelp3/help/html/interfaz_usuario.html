Interfaz de usuario

Esta es la ayuda de la interfaz de usuario.

La aplicación cuenta con una vistosa y eficaz interfaz que nos permite gestionar nuestro sistema de clientes

Sus elementos son:

  • Lista de clientes
  • Botones para realizar acciones sobre clientes:
    • Añadir cliente
    • Modificar cliente
    • Eliminar cliente
  • Botón para salir
