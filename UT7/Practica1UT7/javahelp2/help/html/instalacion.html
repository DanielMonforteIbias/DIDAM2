Instalación

Esta página sirve como guía para la instalación de la aplicación.

En primer lugar debemos obtener el archivo descargándolo de una fuente oficial

Una vez descargado, debemos extraer el archivo zip "aplicacion", y ejecutar el archivo "installer.exe"

Esto abrirá el asistente, donde podremos configurar aspectos como la ruta de instalación y los módulos adicionales
