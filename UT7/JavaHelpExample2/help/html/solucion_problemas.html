Solucion de problemas

Esta pagina sirve como guia para la solucion de problemas.

Algunos de los problemas más comunes con sus soluciones son:

  • Al tratar de añadir un cliente con un correo mal formado, obtendremos el error con codigo 20
  • Al tratar de modificar el correo o el telefono de un cliente, podemos obtener el error con codigo 24 si dichos campos coinciden con los de otro usuario
  • Al tratar de eliminar un cliente, obtendremos el error 11 si este ya ha sido borrado recientemente (ya que existe cierto retraso entre el borrado y la desaparicion del mismo)
